Read Probelm Carefully

Chef lives in a big apartment in Chefland. The apartment charges maintenance fees that he is supposed to pay monthly on time. But Chef is a lazy person and sometimes misses the deadlines. The apartment charges 1000 Rs per month as maintenance fees.The deadline for the N-th month is also over. From his bank statement, he finds the information whether he paid apartment rent for a particular month for not.

Input

First line of the input contains an integer T denoting number of test cases. The description of T test cases follows.

Next line of each test case contains N space separated integers (each of them can be either 0 or 1) denoting whether Chef paid the 1000Rs fees in the corresponding month or not.

Output

For each test case, output a single integer denoting the total amount including fine that Chef has to pay.

Example
Input
4
1 2 3 4

Ouput
0 220 200 100
Languages: C, C++, JAVA, PYTHON, PHP
